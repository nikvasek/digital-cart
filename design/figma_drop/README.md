# Figma drop

Складывайте сюда экспорты из Figma, чтобы сделать **полноценную замену текста** (не накладывание) с сохранением стиля.

## Вариант A (рекомендуемый): фон без текста + шрифты
1) `background-no-text.png` (или `.jpg`)
   - Это тот же макет, но слои **имя / телефон / адрес** скрыты перед экспортом.
   - Экспорт в ширине 375px (1x) и желательно ещё 750px (2x).

2) `typography.json` (можно просто текстом в файле)
   - Для каждого поля (name/phone/address):
     - `fontFamily`, `fontWeight`, `fontSize`, `lineHeight`, `letterSpacing`
     - `color` (hex/rgba)
     - `x`, `y`, `width` (координаты в пределах 375×820 или проценты)
     - `textAlign`

3) Шрифты (если НЕ Google Fonts)
   - Положить в `fonts/` файлы `.woff2` (лучше всего) или `.ttf/.otf`.
   - Важно: только если у вас есть права/лицензия на веб‑использование шрифта.

## Вариант B: SVG с «живым» текстом (подойдёт, если Figma не превращает текст в кривые)
- `template.svg`
- В настройках SVG в Figma выключить **Outline text / Convert text to outlines**.
- Если после экспорта в SVG нет тегов `<text>` (видно только `<path>`), этот вариант не подходит.

## Вариант C: дать ссылку на Figma
- Дайте ссылку на файл/фрейм с доступом (view достаточно), и укажите какие текстовые слои заменяем.

## Как назвать слои в Figma (желательно)
- `card_name`
- `card_phone`
- `card_address`
